## âš¡ Getting Started Locally

To run this project locally, ensure you have [Node.js](https://nodejs.org/) and [npm](https://www.npmjs.com/) installed.

### 1. Clone the repository

```sh
git clone https://github.com/Kgpianghosh006/Whiteboard_frontend.git
cd Whiteboard_frontend
```

### 2. Install dependencies

```sh
npm install
```

### 3. Start the development server

```sh
npm start
```

Visit [http://localhost:3000](http://localhost:3000) to view the app on your browser.
